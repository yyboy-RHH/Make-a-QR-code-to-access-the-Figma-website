# SOP for making a QR code to access the Figma Website

Summary:  
1. 製作 Figma QR Code 簡單快速，關鍵在於確保 Figma 連結正確且分享權限設定為「Anyone with the link can view」。
2. 使用免費 QR Code 工具即可生成，下載後可分享或列印使用。
